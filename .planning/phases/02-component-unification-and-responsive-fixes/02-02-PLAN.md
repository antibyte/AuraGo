## Decisions to honor (from 02-CONTEXT.md):

**D-17:** Use CSS Grid with `auto-fill` and `minmax()` for fluid card layouts
**D-18:** Consistent card gap: 20px across all pages
**D-19:** Cards use `.aura-card` class (from Phase 1 naming convention) for consistent styling

## Unified Grid Pattern:

```css
/* Standard card grid — use this pattern everywhere */
.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
  gap: 20px;
}
```

For status bars (from D-16):
```css
.status-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 20px;
}
```

The difference: `auto-fill` keeps columns even when items are few; `auto-fit` collapses empty columns.
