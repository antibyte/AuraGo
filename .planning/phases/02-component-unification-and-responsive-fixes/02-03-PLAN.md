The shared.css `.toggle` pattern (D-11) is the single source of truth:
```css
/* shared.css checkbox-based toggle */
.toggle {
  appearance: none;
  width: 44px;
  height: 24px;
  background: var(--toggle-off, #555);
  border-radius: 12px;
  cursor: pointer;
  position: relative;
  transition: background 0.2s;
}
.toggle:checked {
  background: var(--toggle-on, var(--accent));
}
.toggle::after {
  content: '';
  position: absolute;
  width: 18px;
  height: 18px;
  background: #fff;
  border-radius: 50%;
  top: 3px;
  left: 3px;
  transition: left 0.2s;
}
.toggle:checked::after {
  left: 23px;
}
```

Replace any config.css toggle customizations with references to the above shared pattern.
