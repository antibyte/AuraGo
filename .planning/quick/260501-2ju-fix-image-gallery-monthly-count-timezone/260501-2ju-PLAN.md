# Quick Task Plan: Fix image gallery monthly count timezone

## Problem

`TestImageGalleryMonthlyCount` fails at the local month boundary because SQLite `CURRENT_TIMESTAMP` stores UTC timestamps while `ImageGalleryMonthlyCount` uses the local month prefix as a string lower bound.

## Scope

- Make monthly image counts compare against UTC bounds derived from the current local calendar month.
- Add regression coverage for local month boundaries and check the ignored insert error in the existing test.
- Validate the focused image-generation tests and affected package tests.
