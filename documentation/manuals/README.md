# AuraGo Manuals

**AuraGo** ist ein vollständig autonomer KI-Agent für Home-Lab-Umgebungen – als einzelnes, portables Binary mit eingebetteter Web-Oberfläche und ohne externe Abhängigkeiten.

Dieser Ordner ist ein Einstiegspfad für Nutzer, die `documentation/manuals` erwarten. Die eigentlichen Handbücher liegen unter `documentation/manual/`.
