Privacy First Policy

No Data Leaves
Your Home Network.

Most automated agents pipe your files, logs, and sensitive directories directly to commercial cloud servers. AuraGo works entirely inside your network. It keeps your API keys secure inside an AES-256 local Vault, operates locally via Ollama or OpenRouter, and executes inside fully sandboxed environments.

✔ End-to-end encrypted Vault for credentials
✔ Secondary security audit model prevents injection
✔ Granular toggles to completely disable dangerous shell actions

// System Protection Layer

# Granular Permission Safety Toggles
DANGER_ZONE:
  ALLOW_SHELL_EXECUTION: false
  ALLOW_PYTHON_SANDBOX: true
  ALLOW_FILE_WRITES: false
  ALLOW_EXTERNAL_NETWORK: true
  ALLOW_SELF_UPDATE: false

Launch in Under Two Minutes

Run our automated installer to verify Docker setups, construct secure directories, and initialize the Setup Wizard.

curl -fsSL https://raw.githubusercontent.com/antibyte/AuraGo/main/install.sh | bash

Once running, navigate to http://localhost:8088 to launch the Setup Wizard!
